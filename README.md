# AI自动阅卷 - Android原生App

基于阿里云百炼平台视觉模型的自动阅卷打分工具。通过屏幕截图+AI评分+自动点击，实现无人值守的批量阅卷。

## 功能特点

- ✅ **API配置**：支持配置百炼API Key和模型ID
- ✅ **自定义截图区域**：自由设置答案区域的坐标
- ✅ **分值按钮配置**：支持添加多个分值按钮的坐标
- ✅ **参考答案输入**：支持输入参考答案，提高评分准确性
- ✅ **自动截图评分**：调用百炼API进行图像分析和评分
- ✅ **自动点击**：根据评分结果自动点击对应分值按钮
- ✅ **循环批阅**：等待页面跳转后自动处理下一份答卷

## 工作原理

```
1. 截图答案区域
2. 连同参考答案一起发送给百炼API
3. 等待AI返回评分
4. 根据评分自动点击对应分值按钮
5. 等待页面自动跳转到下一份答卷
6. 重复上述步骤
```

## 使用步骤

### 1. 配置API
- 打开App，输入百炼API Key
- 输入模型ID（如 qwen-vl-plus）
- 输入参考答案内容
- 点击"保存配置"

### 2. 设置答案区域
- 在对应输入框中填入答案区域的坐标（左、上、右、下）
- 可以使用其他工具辅助获取屏幕坐标

### 3. 配置分值按钮
- 点击"添加"按钮
- 输入分值和该分值按钮的坐标
- 支持添加多个分值按钮
- 可点击"测试点击"验证是否配置正确

### 4. 开启无障碍服务
- 进入手机设置 → 无障碍
- 找到"AI自动阅卷"服务并开启

### 5. 开始阅卷
- 打开阅卷系统网页或应用
- 点击"开始阅卷"
- 授予屏幕录制权限
- 系统自动开始批阅

## 注意事项

1. 需要阿里云百炼API Key和有效的模型ID
2. 首次使用需要授权屏幕录制权限
3. 必须开启无障碍服务才能实现自动点击
4. 需要准确配置答案区域和分值按钮的坐标
5. 建议在稳定的网络环境下使用
6. 本工具仅供学习和研究使用
